# Dynamica Plantillas

Repositorio de plantillas dinámicas para correo electrónico.

## Estructura

- `templates/` — plantillas de correo organizadas por formato.
  - `html/` — plantillas en HTML.
  - `txt/` — plantillas en texto plano.
- `scripts/` — utilidades para renderizar y gestionar plantillas.

## Uso

Pendiente de definir según las necesidades del proyecto.
